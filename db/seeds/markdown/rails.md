# Ruby on Rails

## Active Record

ORM responsável pela comunicação com o banco de dados.

## N+1 Query

Problema causado por consultas repetidas. Pode ser mitigado com includes.

## Sidekiq

Ferramenta para processamento assíncrono baseada em Redis.

## Service Objects

Classes responsáveis por encapsular regras de negócio complexas.

## Active Job

Camada de abstração para filas de processamento.
