# Perguntas Frequentes

## O que é Duck Typing?

Conceito onde o comportamento do objeto importa mais que sua classe.

## Diferença entre Proc e Lambda?

Lambda valida argumentos e possui comportamento semelhante a métodos.
Proc é mais flexível.

## O que é N+1 Query?

Múltiplas consultas desnecessárias geradas pelo ORM.

## Como evitar N+1?

Utilizando eager loading através de includes.

## Quando usar Sidekiq?

Processamento assíncrono de tarefas demoradas.
